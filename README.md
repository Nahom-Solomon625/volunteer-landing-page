# BrightPath Volunteers Landing Page

A modern, eye-catching landing page for a volunteer organization built with **only HTML + CSS** (no JS, no frameworks).

## What's Included

- Sticky navigation + hero section
- Featured opportunities panel
- Stats + programs grid
- Impact snapshot card
- Upcoming events list
- Testimonials
- Volunteer sign-up form (mock/demo form)
- FAQ accordion (native `<details>` / `<summary>`)
- Final call-to-action + footer
- Responsive layout + basic accessibility (skip link, focus styles)

## Run / View

1. Open `index.html` in your browser.

Optional (Windows PowerShell):

```powershell
start .\index.html
```

## Files

- `index.html` - Page structure + mock content
- `style.css` - Full styling (gradients, layout, responsiveness)

## Notes

- All content is mock/demo data.
- Form buttons/links do not submit anywhere (no backend).
